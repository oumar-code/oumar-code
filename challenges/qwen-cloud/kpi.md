# Qwen Cloud — KPI Emphasis

## Most Important Metrics (in priority order)

1. **`quality_score`** — Multi-language correctness is the headline KPI.
2. **`reliability`** — Cloud-native story requires near-perfect uptime.
3. **`avg_latency_ms`** — Competitive with closed models on standard hardware.
4. **`cost_per_run`** — Open-weight advantage: cheaper than GPT-4 at same quality.

## Threshold Targets

| Metric | Minimum Passing | Gold Target |
|---|---|---|
| quality_score | ≥ 0.80 | ≥ 0.92 |
| reliability | ≥ 0.95 | ≥ 0.99 |
| avg_latency_ms | ≤ 2000 | ≤ 800 |
| cost_per_run | ≤ $0.10 | ≤ $0.03 |

## Languages to Demo
- English (baseline)
- French (West Africa coverage)
- Arabic (MENA coverage)
- Swahili (East Africa coverage)
- Mandarin (Qwen home turf)

## Story Framing
Lead with **multilingual reach**: "One model, five languages, zero fine-tuning."
